用python3写的cgi服务器脚本无法显示中文

2018-11-28

在脚本前加上如下两句即可正常显示中文

import codecs, sys
sys.stdout = codecs.getwriter('utf8')(sys.stdout.buffer)

脚本示例：

#!/miniconda3/bin/python

import codecs, sys
sys.stdout = codecs.getwriter('utf8')(sys.stdout.buffer)

print ("Content-type:text/html")
print ()
print ('')
print ('')
print ('')
print ('Hello Word')
print ('')
print ('')
print ('
Hello Word!中文测试
')
print ('')
print ('')
